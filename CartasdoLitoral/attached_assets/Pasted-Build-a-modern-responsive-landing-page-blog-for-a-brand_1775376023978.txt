Build a modern, responsive landing page + blog for a brand called "Cartas do Litoral."

// ─────────────────────────────────────────
// DESIGN DIRECTION
// ─────────────────────────────────────────

This is a calm, editorial, coastal storytelling brand.
The design should feel: minimal, airy, emotional, premium but simple,
inspired by print (magazine / letter aesthetic).

Avoid anything that feels: tech startup, loud, overdesigned, cartoonish.

Do NOT use any external UI component libraries (no shadcn, MUI, Chakra).
Build all components from scratch using Tailwind CSS only.
Keep the component tree clean and flat.

// ─────────────────────────────────────────
// HERO SECTION
// ─────────────────────────────────────────

Layout: Split screen on desktop.
LEFT:  Full-height lifestyle image (use uploaded PNG)
RIGHT: Textured paper background (#F5F1EA) with content

LEFT IMAGE:
Coastal Brazil. Soft, slightly muted tones.
Woman walking along shoreline.
Bottom of dress slightly wet.
Calm, reflective mood.

RIGHT SIDE CONTENT:
Headline (large, Playfair Display, elegant):
  CONTINUE A HISTÓRIA
  COM A MEL

Note: "Mel" is the narrator's nickname — warm, Brazilian, intimate.
It should feel like the reader already knows her.

Subtext (3 lines, Lora, smaller):
  Uma carta por mês
  Um novo lugar do litoral
  Para ler com calma.

Decorative elements (TWO elements — both required):

  1. POSTAL WAVE LINES
     Thin, minimal horizontal wave lines
     Style: classic postal cancellation marks
     Color: #6F8A7E at 30% opacity
     Position: subtle, behind or near the headline
     SVG drawn, not an image

  2. VINTAGE STAMP
     Position: top-right corner of the right panel
     OR overlapping slightly between left image and right panel
     Style: rectangular with perforated/dotted border edge
     Inside the stamp:
       Small coastal illustration (wave or lighthouse silhouette)
       Text: "CARTAS DO LITORAL"
       Text: "BRASIL" at bottom
       Faint number: "001" like a postage value
     Color: #A67C52 tones on #EAE3D8 background
     Slightly rotated (2-3 degrees)
     Aged/worn feel — slight opacity reduction (85%)
     SVG drawn, not an image
     Do NOT make it too large — it is a detail, not a hero element

CTA button:
  Text:  "Começar com a primeira carta"
  Color: #A67C52
  Style: Soft outlined button, generous padding

Visual restraint:
  Include 1-2 Polaroid-style images (slightly rotated)
  Keep generous whitespace. Do NOT overcrowd.
  The stamp and wave lines are details — they should reward
  close attention, not dominate the layout.
